# AI Engineering Learning Roadmap

Based on *AI Engineering* by Chip Huyen (O'Reilly, 2024).

Each chapter file contains:
- A short summary of the concept
- Practical tasks that teach it AND improve CuentaCuentos
- A checkbox list to track progress

Work through them in order — each chapter builds on the previous.
